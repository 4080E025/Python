# For 迴圈(loop)
```
for name in ['orange', 'apple']:
	print(name)

答案:
orange
apple
```

```
for index in range(0, 6):
	print(index)

答案:
0
1
2
3
4
5
```
```
sum=0

for x in range(1,101):
    sum +=x
    
print(sum)

答案:5050(1+2+3+...+101)
```
```
sum=0

for x in range(1,101,2):
    sum +=x
    
print(sum)

答案:2500(1+3+5+7+....+101)
```
```
names = ['christopher','Susan']
index = 0
while index < len(names):
  print(names[index])
 #Change the condition!!
  index = index+1

答案:
christopher
Susan
```
